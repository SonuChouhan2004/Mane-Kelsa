# Mane-Kelsa

## Android App Development using GenAI

Mane-Kelsa is a hyper-local Android application designed to connect domestic workers, gardeners, and local residents in small towns.

The app acts as a digital local job board where workers can create profiles, update their daily availability, and receive nearby work opportunities.

---

## Features

- Worker Profile Creation
- Daily Availability Toggle
- One-Tap Calling Feature
- Firebase Realtime Database Integration
- Kannada Localized UI
- Rating System for Workers

---

## Technologies Used

- Android Studio
- Kotlin
- Firebase Realtime Database
- XML UI Design

---

## Problem Statement

Domestic workers and gardeners in small towns often depend on word-of-mouth to find work opportunities. Mane-Kelsa helps digitize the local labor market and reduce unemployment gaps.
